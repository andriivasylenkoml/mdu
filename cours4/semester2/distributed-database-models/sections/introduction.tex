\section*{МЕТА РОБОТИ}
\phantomsection
\addcontentsline{toc}{section}{МЕТА РОБОТИ}

Метою лабораторної роботи є дослідження принципів керування транзакціями у розподіленому середовищі та практична реалізація банківського переказу між двома вузлами бази даних PostgreSQL із застосуванням двофазного протоколу фіксації (2PC).

Для досягнення поставленої мети необхідно виконати такі завдання:
\begin{itemize}
    \item засвоїти принципи керування транзакціями у розподіленому середовищі;
    \item розглянути механізми забезпечення узгодженості даних;
    \item порівняти протоколи фіксації транзакцій;
    \item оцінити ризики паралельного доступу та відмов вузлів;
    \item реалізувати модель переказу коштів між двома вузлами PostgreSQL із використанням 2PC.
\end{itemize}
